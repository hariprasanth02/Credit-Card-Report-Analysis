## 📌 Project Overview

This project focuses on analyzing **credit card usage patterns**, revenue generation, and customer demographics using Power BI.

It consists of two main dashboards:
- 📊 **Transaction Analysis**
- 👥 **Customer Insights**

## 📊 Dashboard Pages

### Page 1 — Transaction Report

Provides insights into revenue, spending behavior, and transaction trends.

| Section | Details |
|--------|--------|
| KPI Cards | Revenue, Interest, Transactions, Amount |
| Quarter Analysis | Revenue trend across Q1–Q4 |
| Card Category | Blue, Silver, Gold, Platinum performance |
| Expenditure Type | Bills, Entertainment, Fuel, Grocery |
| Usage Mode | Swipe, Chip, Online |

#### 🔍 Key Insights:
- Q3 recorded the **highest revenue (~14.2M)**
- **Swipe transactions dominate (35M)** usage
- **Blue card users generate the most revenue (~46M)**
- Bills and Entertainment are top spending categories
- Digital transactions (Chip & Online) are underutilized

---

### Page 2 — Customer Report

Focuses on customer demographics, income, and behavioral insights.

| Section | Details |
|--------|--------|
| KPI Cards | Revenue, Interest, Income, CSS |
| Gender Analysis | Revenue comparison (Male vs Female) |
| Age Group | Distribution across age segments |
| Customer Job | Businessman, White-collar, Govt, etc. |
| Salary Group | High, Medium, Low income |
| Top States | TX, NY, CA |

#### 🔍 Key Insights:
- **Male customers contribute more revenue (~30M)** than females
- **Businessmen are the top revenue contributors (~17M)**
- High-income group dominates spending (~22M)
- Married customers spend more than single customers
- Revenue is concentrated in a few high-value segments

---

## 📈 Business Insights

- 🎯 Revenue is heavily driven by **Blue card users and high-income groups**
- 💡 **Q3 seasonal spike** suggests increased spending cycles
- 📉 **Online & Chip transactions are low**, indicating digital adoption opportunity
- 🧠 Customer targeting should focus on:
  - High-income professionals
  - Graduate-level customers
  - Business-related occupations
- 💳 Customer acquisition cost is highest for premium users → optimize marketing ROI

## 🚀 How to Use

1. Clone the repository:
```bash
git clone https://github.com/your-username/credit-card-dashboard.git
```

2. Open `CreditCardDashboard.pbix` in Power BI Desktop

3. Update dataset paths if needed (`credit_card.csv`, `customer.csv`)

4. Click **Refresh**

5. Explore the dashboard with filters and slicers
